| 功能 | 说明 |

| 📖 **书籍搜索** | 支持书名、作者、ISBN 等多种搜索方式 |

## 🚀 快速开始

### 环境要求

- Flutter SDK 3.8+
- Android Studio / VS Code
- Android 设备或模拟器

### 安装步骤

```bash
# 克隆仓库
git clone https://github.com/shiyi-0x7f/olib-mobile.git

# 进入项目目录
cd olib-mobile

# 安装依赖
flutter pub get

# 运行应用
flutter run
```

### 构建 APK

```bash
flutter build apk --release
```

## 🏗️ 项目结构

```
lib/
├── l10n/           # 多语言文件 (16+ 种语言)
├── models/         # 数据模型
├── providers/      # 状态管理 (Riverpod)
├── routes/         # 路由导航
├── screens/        # 界面
├── services/       # API 和存储服务
├── theme/          # 主题配置
└── widgets/        # 可复用组件
```

## 🛠️ 技术栈

- **框架**: Flutter
- **状态管理**: Riverpod
- **本地存储**: Hive
- **网络请求**: http 包
- **多语言**: 16+ 种语言
